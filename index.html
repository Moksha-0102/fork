Fork() is a spatial text editor for mapping out logic trees, user flows, and branching thoughts.

🧠

The Use Cases

Standard docs force you to think top-to-bottom. Use this essentially infinite canvas to brainstorm app architectures, draft branching narratives, or untangle complex decisions.

⌨️

Zero-Mouse Flow

Never break your flow state. Press Cmd/Ctrl + ↵ to branch logic downward, and Cmd/Ctrl + B to spawn sibling thoughts side-by-side and more keyboard shortcuts are available to make sure you never get interrupted.

⚡

Ship Your Logic

Everything auto-saves locally. When your thought tree is complete, use the 1-click exporter to generate a clean Markdown list, JSON structure, or a PNG.
