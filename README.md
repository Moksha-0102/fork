## ⚡ Quick Start

**Fork() is a client-side web application.** To try it immediately: **[Open the Live Demo](https://fork-self.vercel.app/)**

*(No sign-ups, no databases. Your data is stored locally in your browser.)*

---

## ✨ Features

* **Zero-Mouse Flow:** Spawn child branches (`Cmd` + `Enter`), create siblings (`Cmd` + `B`), and utilize many other keyboard shortcuts to ensure you never have to take your hands off your keyboard.
* **Focus Mode:** Dim the canvas and fade inactive branches to stay locked into your current train of thought.
* **Export your data:** 1-click export your map into a formatted Markdown list, a JSON object, PNG, or SVG.
* **Local Persistence:** Everything auto-saves natively to your browser's `localStorage`. You never lose a thought.
* **Dark Mode:** Customize your theme to work comfortably at any time fo the day/

## 💻 How to run it locally

Fork() requires zero build steps or package managers.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Moksha-0102/fork.git](https://github.com/Moksha-0102/fork.git)
    cd fork
    ```

2.  **Serve the directory:**
    You can use any local web server. For example, using Python 3:
    ```bash
    python -m http.server 3000
    ```

3.  **Launch the app:**
    Open `http://localhost:3000` in your web browser.
